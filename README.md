# AcademicDressSystem
學位服系統

## Project
A Python-based desktop management system for handling academic gown borrowing and returning records.

## Features
- Student ID search
- Borrow / return management
- Return status tracking (returned, lost, damaged)
- Excel report generation
- Daily return statistics
- Unreturned gown report
- Department and college statistics

## Tech Stack
- Python
- Tkinter (GUI)
- OpenPyXL (Excel processing)
